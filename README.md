# 미션 - 구현 목록

1. 랜덤한 숫자 생성 발생 : generateNumber
    - 세 자리 수이며 각 자리수가 1이상 9이하인 난수 생성

2. 입력의 유효성 확인 : checkInput
    - 게임이 끝나지 않았을 때의 입력인 경우, 입력이 세 자리인지, 세 자리 모두 숫자인지, 중복은 없는지 확인

3. 스트라이크 계산 : calculateStrike

4. 볼 계산 : calculateBall
